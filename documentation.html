Textboard Documentation

Classes

Board()

When Board() class in initialized, it creates Board().catalog - an empty dictionary, and an integer idCounter=0
Threads are stored as a (threadID:title) pair in catalog

.create(title)

Creates a thread using a title that was passed as an argument

.show(amount)

Prints a specified amount of threads, if no argument was given or the argument was higher than the number of threads, method prints all threads

.delete(threadID)

If threadID exists in Board().catalog, it sets the thread value to False, which prevents .show() method from printing it among other threads

Thread(board)

Thread() class is initialized with a board instance, Thread() class sets Thread().board to the passed board instance
When initialized, Thread() class creates replyID=0 integer, and replies dictionary
Replies are stored in Thread().replies in a specific way: (replyID:[threadID, replyMsg])

.reply(threadID, replyMsg)

The method makes a new item in Thread().replies using a unique replyID (which then is incremented by one), threadID and replyMsg (which is the actual reply string)

.showReplies(threadID)
